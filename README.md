This is small "Space Shooter" game inspired by the Arcade game, Space Invaders. 
Right now the game runs forever, meaning that the once the player destroys all 
alien ships, the game restarts, or when any of the alien's ships collide (touch) 
with the player's ship, the game restarts. 

My next goal is to add an "Game Over" screen, and a score counter.

I got the images from https://pixabay.com/

You can start the game by running the "space_shooter.py" python file.
